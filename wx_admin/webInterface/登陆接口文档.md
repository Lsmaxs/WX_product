# 登录接口文档  

## 1.身份验证，令牌（TOKEN）获取

> 用于获取UC TOKEN

### 接口：

 GET   
>  /uww-pro.qtonecloud.cn/v2/oauth/token

### 请求参数：

| 名称 | 类型 | 定义 | 必需 | 默认值 | 说明
---|---|---|---|---|---|
| product | string | 查询关键词   |     | weixiao | 默认微校
| client_id | string | 页码        |     | weixiao | 默认微校
| client_secret | string | 角色        |     | weixiao | 默认微校
| grant_type | string | 授权类型 |     | password| 默认密码
| scope | string | 范围权限     |    | read  | 只读
| username | string | 用户名     |  true  | ""  | 用户名
| password | string | 用户密码     |  true  | ""  | 用户密码

### 响应：

>  响应格式：JSON

#### 成功：200 
```
{
    bizData: {
           "additionalInformation":{},
           "expiration":{string},//2016-12-09 21:17:43
           "expired":{boolean},//false
           "expiresIn":{number},//43199
           "refreshToken":{
                "expiration":{string}, //2016-12-22 09:22:59
                "value": {string}
           },
           "scope":[string],//read
           "tokenType":{string},//bearer
           "value":{string}//4a19212d-f632-4f1c-af9f-216a9716ac6c
        },
    rtnCode:"0000000",//返回码状态
    ts：{number}//时间码？
}

```

#### 参数不合法：400

    {
        
    }
